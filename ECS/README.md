AWS Elastic Container Service

Components involved:
- IAM Roles: for Container Instance, for Task Execution
- VPC
- Availability Zone
- Subnet
- Security Groups
- EC2 (ECS-optimized Amazon Linux image)
- Application Load Balancer
- Target Group
- Launch Configuration
- Auto Scaling Group
- Route 53 (Optional)
- ECR
- ECS Cluster
- ECS Task Definition
- ECS Service
- ECS Fargate (Optional)
